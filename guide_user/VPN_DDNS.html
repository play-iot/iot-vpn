VPN DDNS k8s job

Table of Contents

Query VPN client session based on hostname, vpn ip address then create DNS record in Cloud DNS. The job is setup to run on k8s cronjob

  • Use this utility

  • Cronjob manifest:

    • 10_create-config-secret.sh Create config/secret

    • 11_vpn-dns-sync-cronjob.yaml

Deployment steps

  • Prepare google-service-account.json with right permission to update Cloud DNS records.

  • Copy and fill information based on a customer code. For example: playio is customer code

    • customer.env.tmpl = playio.env

    • customer.secret.env.tml = playio.secret.env

    • .runner.env.tmpl = .prod.env

  • Checkout utility

PROJECT_DIR=<your-path>
python deploy.py -e prod -m gcp -d $PROJECT_DIR/cli/k8s/ddns
